# removeAttribute

`removeAttribute(node, name)` is the platform `removeAttribute` on
`node`. See also [`setAttribute`](/use/dom/setAttribute) /
[`hasAttribute`](/use/dom/hasAttribute). Prefer JSX attributes — this
is an imperative escape hatch. Part of [`pota/use/dom`](/use/dom).
