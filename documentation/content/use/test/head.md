# head

`head()` returns `document.head.innerHTML.trim()` — a quick snapshot
of the document head for assertions. The body counterpart is
[`body`](/use/test/body). Part of [`pota/use/test`](/use/test).

**Returns:** the trimmed `document.head.innerHTML` string.
