# body

`body()` returns `document.body.innerHTML.trim()` — a quick snapshot
of the rendered body for assertions. The head counterpart is
[`head`](/use/test/head). Part of [`pota/use/test`](/use/test).

**Returns:** the trimmed `document.body.innerHTML` string.
