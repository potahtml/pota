# isHash

`isHash(url)` returns `true` when `url` starts with `#`. Part of
[`pota/use/url`](/use/url).

## Arguments

| Argument | Type     | Description       |
| -------- | -------- | ----------------- |
| `url`    | `string` | The URL to check. |

**Returns:** `boolean` — `true` when `url` starts with `#`.
