# capitalizeFirstLetter

`capitalizeFirstLetter(s)` uppercases the first character; the rest is
untouched. Part of [`pota/use/string`](/use/string).

## Arguments

| Argument | Type     | Description               |
| -------- | -------- | ------------------------- |
| `s`      | `string` | The string to capitalize. |

**Returns:** the input string with its first character uppercased.
